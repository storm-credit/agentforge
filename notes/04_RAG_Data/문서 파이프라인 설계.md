# 문서 파이프라인 설계

## 흐름

```text
문서 수집
-> 원본 저장
-> 파싱
-> 텍스트 정규화
-> chunking
-> metadata/ACL 부여
-> embedding
-> vector index
-> 검색
-> rerank
-> citation 답변
```

## MVP 지원 형식

- PDF
- DOCX
- XLSX
- TXT
- Markdown

## chunk metadata

- document_id
- source_id
- title
- page
- section
- department_scope
- confidentiality_level
- created_at
- indexed_at
