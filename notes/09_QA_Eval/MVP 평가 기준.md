# MVP 평가 기준

## 기능 기준

- 관리자가 문서 기반 에이전트를 생성할 수 있다.
- 사용자가 질문하면 권한 있는 문서만 검색된다.
- 답변에는 근거 문서 citation이 포함된다.
- 실행 단계가 run, run_step, audit_event로 남는다.

## 품질 기준

| 기준 | 목표 |
|---|---|
| 권한 위반 | 0건 |
| citation 포함률 | 95% 이상 |
| 유용 답변 비율 | 80% 이상 |
| 평균 응답시간 | MVP 기준 10초 이내 목표 |
| 회귀 테스트 | 주요 질문 30~50개 |

## 실패로 보는 경우

- 권한 없는 문서가 답변 근거로 사용됨
- citation 없이 단정적인 답변 제공
- 모르는 내용을 근거 없이 생성
- 민감정보가 마스킹 없이 노출
