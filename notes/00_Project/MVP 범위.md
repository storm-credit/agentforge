# MVP 범위

## MVP 이름

사내 문서 기반 RAG 에이전트 빌더

## 포함 범위

- 문서 업로드 또는 수집 등록
- PDF, DOCX, XLSX, TXT, Markdown 기본 파싱
- chunking, embedding, vector index 생성
- 사용자/부서/문서등급 기반 검색 필터
- 근거 citation이 포함된 답변
- 에이전트 카드 등록과 버전 관리
- 실행 로그, 검색 문서, 답변 로그 저장
- 관리자용 최소 Agent Studio 화면

## 제외 범위

- ERP/그룹웨어 쓰기 작업
- 메일 자동 발송
- 결재 자동 처리
- 전사 모든 문서 연결
- 완전 자율 멀티 에이전트
- 고급 비용 최적화와 자동 모델 라우팅

## MVP 성공 기준

- 권한 없는 문서 노출 0건
- 답변의 근거 citation 포함률 95% 이상
- 샘플 질문 기준 정답/유용 답변 비율 80% 이상
- 주요 실행 단계가 audit log에 남을 것
- 관리자 1명이 새 문서 에이전트를 설정할 수 있을 것
