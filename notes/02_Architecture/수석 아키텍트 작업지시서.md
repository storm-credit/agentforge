# 수석 아키텍트 작업지시서

## 역할

수석 아키텍트는 Agent Forge의 전체 시스템 구조, 모듈 경계, 데이터 흐름, 배포 단위를 설계한다.

## 설계 원칙

- Control Plane, Runtime Plane, Data Plane을 분리한다.
- 보안과 감사 로그를 핵심 기능으로 본다.
- 외부 SaaS 의존 없이 사내망에서 동작 가능한 구조를 우선한다.
- MVP는 작게 만들되, DB/ERP/그룹웨어 Tool Pack 확장을 막지 않는다.
