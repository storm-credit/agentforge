# Agent Forge Project Overview

Agent Forge는 사내망 환경에서 승인된 모델, 사내 문서, 업무 시스템 도구, 권한정책, 감사 로그를 조합해 업무용 AI 에이전트를 생성하고 운영하기 위한 내부 Agent Builder 플랫폼이다.

## 한 줄 정의

사내망에서 현업/관리자가 업무용 AI 에이전트를 안전하게 만들고 운영할 수 있는 통제형 Agent Builder.

## 1차 MVP

1차 MVP는 사내 문서 기반 RAG 에이전트 빌더다.

- 문서 수집과 인덱싱
- ACL-aware retrieval
- 근거 citation 답변
- Agent Build 설정과 버전 관리
- 실행 로그와 감사 로그
- 최소 Agent Studio UI
