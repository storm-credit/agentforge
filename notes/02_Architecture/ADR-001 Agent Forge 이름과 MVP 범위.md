# ADR-001 Agent Forge 이름과 MVP 범위

## 상태

Accepted

## 결정

프로젝트 이름은 Agent Forge로 한다. 1차 MVP는 사내 문서 기반 RAG 에이전트 빌더로 제한한다.

## 배경

사내망 환경에서는 외부 AI 서비스와 SaaS형 Agent Builder를 그대로 사용하기 어렵다. 따라서 Craft AI 같은 개념은 참고하되, 내부 모델, 내부 문서, 내부 권한정책을 사용하는 통제형 플랫폼이 필요하다.

## 결과

- 제품명과 저장소명을 Agent Forge로 통일한다.
- 처음부터 DB/ERP/그룹웨어 자동화를 포함하지 않는다.
- 문서 검색, 권한 필터, 근거 답변, 실행 로그를 먼저 검증한다.
