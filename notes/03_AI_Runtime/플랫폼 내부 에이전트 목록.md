# 플랫폼 내부 에이전트 목록

| 에이전트 | 역할 | MVP 포함 |
|---|---|---|
| Planner Agent | 사용자 요청을 처리 단계로 분해 | 예 |
| Retriever Agent | 권한 기반 문서 검색 | 예 |
| Security Guard Agent | 정보유출, prompt injection, 권한 위반 점검 | 예 |
| Critic Agent | 답변 근거와 품질 검토 | 예 |
| Tool Executor Agent | 사내 API/DB/업무 도구 호출 | 이후 |
| Human Approval Agent | 위험 작업 승인 요청 생성 | 이후 |
| Memory Agent | 업무 컨텍스트 저장 기준 관리 | 이후 |
| Cost/Latency Agent | 모델 선택과 응답 비용 최적화 | 이후 |
