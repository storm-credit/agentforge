# 권한 ACL Matrix

| 대상 | 권한 축 | MVP 처리 | 비고 |
|---|---|---|---|
| 사용자 | user_id | 필수 | 실행 로그와 연결 |
| 부서 | department_id | 필수 | 문서 검색 필터 |
| 직급/역할 | role | 선택 | 승인 정책에 사용 |
| 문서 등급 | confidentiality_level | 필수 | 공개/내부/제한/기밀 |
| 프로젝트 | project_scope | 이후 | 부서 간 프로젝트용 |

## 문서 등급 초안

| 등급 | 설명 | MVP 정책 |
|---|---|---|
| 공개 | 전 임직원 열람 가능 | 검색 가능 |
| 내부 | 사내 구성원 열람 가능 | 검색 가능 |
| 제한 | 특정 부서/역할만 열람 가능 | ACL 필수 |
| 기밀 | 제한된 사용자만 열람 가능 | MVP 제외 또는 별도 승인 |
