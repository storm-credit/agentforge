# 프론트엔드 전문가 작업지시서

## 역할

프론트엔드 전문가는 Agent Studio UI, 관리자 콘솔, 실행 추적 화면, 승인 콘솔의 사용자 흐름을 설계한다.

## 주요 질문

- 현업 사용자는 어디까지 설정할 수 있고, 관리자는 어디부터 승인해야 하는가?
- Agent Builder는 wizard형인가, form형인가?
- Run Trace Viewer는 어떤 단계 정보를 보여줘야 하는가?
- 문서 권한 오류나 citation 누락을 어떻게 표시할 것인가?
