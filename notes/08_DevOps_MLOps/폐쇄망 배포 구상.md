# 폐쇄망 배포 구상

## 기본 구성

- Backend: FastAPI
- Frontend: Next.js
- Metadata DB: PostgreSQL
- Vector DB: Qdrant 또는 pgvector
- Object Storage: MinIO
- Model Serving: vLLM 우선, 경량은 Ollama
- Observability: OpenTelemetry, Prometheus, Grafana
