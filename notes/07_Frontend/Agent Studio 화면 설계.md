# Agent Studio 화면 설계

## MVP 화면

| 화면 | 목적 |
|---|---|
| Agent 목록 | 생성된 에이전트 확인 |
| Agent Builder | 이름, 목적, 모델, 문서, 권한 설정 |
| Knowledge Source | 문서 소스 등록과 인덱싱 상태 확인 |
| Test Chat | 에이전트 테스트 |
| Run Trace Viewer | 검색 문서, 단계, 답변 로그 확인 |
| Admin Settings | 모델, 권한, 기본 정책 관리 |

## Builder 흐름

1. 에이전트 이름과 목적 입력
2. 모델 선택
3. 지식 소스 선택
4. 권한 정책 선택
5. 답변 스타일과 citation 정책 선택
6. 테스트 질문 실행
7. 저장 또는 배포
