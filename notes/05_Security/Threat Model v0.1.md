# Threat Model v0.1

## 주요 위협

| 위협 | 시나리오 | 대응 |
|---|---|---|
| 권한 없는 문서 노출 | 검색 결과에 타부서 문서가 포함됨 | 검색 전 ACL filter 강제 |
| Prompt injection | 문서 안의 악성 지시가 모델 행동을 바꿈 | 문서 지시 무시 정책, Security Guard |
| 민감정보 출력 | 개인정보/기밀 문장이 답변에 포함됨 | PII 탐지와 masking |
| 로그 과다 저장 | 민감 원문이 로그에 그대로 저장됨 | hash, 요약, 최소 저장 |
| 도구 오남용 | 에이전트가 쓰기 API를 실행함 | MVP는 읽기/검색 중심, 쓰기 작업 승인 |
