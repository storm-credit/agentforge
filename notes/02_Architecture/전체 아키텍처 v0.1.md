# 전체 아키텍처 v0.1

## 구조

```mermaid
flowchart TB
  U["사용자/관리자"] --> UI["Agent Studio"]
  UI --> CP["Control Plane"]
  UI --> RT["Runtime Plane"]

  CP --> AR["Agent Registry"]
  CP --> TR["Tool Registry"]
  CP --> PE["Policy Engine"]
  CP --> AU["Audit Log"]

  RT --> OR["Runtime Orchestrator"]
  OR --> PL["Planner Agent"]
  OR --> RA["Retriever Agent"]
  OR --> SG["Security Guard"]
  OR --> CR["Critic Agent"]

  RT --> DP["Data Plane"]
  DP --> DOC["Document Store"]
  DP --> VDB["Vector DB"]
  DP --> ACL["Document ACL"]

  RT --> MG["Model Gateway"]
  MG --> LLM["Local LLM"]
  MG --> EMB["Embedding Model"]
  MG --> RR["Reranker"]
```

## Control Plane

에이전트 생성, 버전 관리, 도구 등록, 권한 정책, 감사 정책을 관리한다.

## Runtime Plane

사용자 요청을 받아 계획, 검색, 답변 생성, 검토, 보안 점검, 로그 저장을 실행한다.

## Data Plane

문서, chunk, vector index, ACL, 원본 파일 메타데이터를 관리한다.
