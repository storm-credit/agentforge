# 보안 아키텍트 작업지시서

## 역할

보안 아키텍트는 폐쇄망 제약, 접근제어, 감사로그, 민감정보 통제, prompt injection 방어 기준을 정의한다.

## 주요 질문

- 외부망 연동이 완전히 차단되는가, 제한 게이트웨이가 있는가?
- 사용자 인증은 AD, LDAP, OIDC 중 무엇인가?
- 문서 등급과 부서 권한은 어디에서 가져오는가?
- 어떤 요청이 human approval을 요구하는가?
