# 설명 스크립트

## 10초 설명

Agent Forge는 사내망에서 업무용 AI 에이전트를 안전하게 만들고 운영하는 내부 Agent Builder입니다.

## 30초 설명

Agent Forge는 Craft AI 같은 에이전트 빌더 개념을 사내망 제약에 맞게 재구성한 플랫폼입니다. 외부 AI 서비스를 그대로 쓰기 어려운 환경에서 사내 문서, 내부 모델, 업무 시스템 도구, 권한정책을 조합해 업무용 AI 에이전트를 만들고, 실행 기록과 보안 통제를 함께 관리합니다.

## 2분 설명

이 프로젝트의 목표는 단순 챗봇을 만드는 것이 아니라, 사내에서 반복적으로 업무용 에이전트를 만들 수 있는 표준 플랫폼을 만드는 것입니다. 1차 MVP는 문서 기반 RAG 에이전트 빌더입니다. 사용자는 승인된 문서 묶음과 모델을 선택하고, 부서 권한과 답변 정책을 설정해 보안 규정 상담, 업무 매뉴얼 질의응답 같은 에이전트를 만들 수 있습니다. 모든 검색 문서, 답변, 실행 단계는 감사 로그로 남기며, 향후 DB 조회, 그룹웨어, ERP 연동 에이전트로 확장합니다.

## 회의용 마무리 문장

처음부터 전사 자동화를 만들지 않고, 문서 기반 에이전트 빌더 MVP로 권한, 근거 답변, 실행 로그를 검증한 뒤 시스템별 Tool Pack을 확장하겠습니다.
